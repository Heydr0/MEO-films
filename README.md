# MEO-films

**Автор:** Рахман Платон Евгеньевич

**Язык:** Python 3.14

## Описание

Простое приложение для хранения личной кинотеки. Позволяет добавлять фильмы с указанием названия, жанра, года выпуска и рейтинга. Реализована фильтрация по жанру и году. Данные сохраняются в файл `movies.json`.

## Использование

1. Установите Python 3.14.
2. Запустите `main.py`.
3. Добавляйте фильмы через поля ввода.
4. Используйте поля фильтрации для поиска.
5. Данные сохраняются автоматически.
